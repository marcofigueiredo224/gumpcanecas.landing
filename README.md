# Gump Canecas — Landing Page

## Como usar
1. Abra a pasta e dê duplo clique no `index.html` (ou use Live Server no VS Code).
2. Edite o número do WhatsApp em `script.js`:
   - `const WHATSAPP_NUMBER = "55SEUNUMEROAQUI";`
   - Coloque somente dígitos: país + DDD + número.
3. Para trocar produtos, edite o array `PRODUCTS` em `script.js`.
4. Substitua as imagens em `assets/` pelas fotos reais das canecas.

## Observação
O WhatsApp não tem carrinho nativo. Aqui o "carrinho" vira uma mensagem pré-preenchida com os itens + observações.
